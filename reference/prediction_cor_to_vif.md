# Prediction of the model `gam_cor_to_vif` across correlation values

Dataframe with predicted VIF threshold corresponding to a given
correlation threshold..

## Usage

``` r
data(prediction_cor_to_vif)
```

## Format

A dataframe with 901 rows and 2 numeric columns:

- max_cor:

  Maximum allowed pairwise correlation, from 0.10 to 1.00 in steps of
  0.001.

- max_vif:

  Predicted VIF threshold corresponding to each `max_cor`.

## Details

Values were generated by applying
[`mgcv::predict.gam()`](https://rdrr.io/pkg/mgcv/man/predict.gam.html)
to the fitted model
[gam_cor_to_vif](https://blasbenito.github.io/collinear/reference/gam_cor_to_vif.md)
and rounding to three decimal places.

## Examples

``` r
data(prediction_cor_to_vif)
head(prediction_cor_to_vif)
#>   max_cor max_vif
#> 1   0.100   1.095
#> 2   0.101   1.096
#> 3   0.102   1.097
#> 4   0.103   1.099
#> 5   0.104   1.100
#> 6   0.105   1.101
plot(max_vif ~ max_cor, data = prediction_cor_to_vif, type = "l")
```
