# Small example dataframe

Same as dataset
[vi](https://blasbenito.github.io/collinear/reference/vi.md), but with
only 610 rows. Used in code examples to shorten runtime.

## Usage

``` r
data(vi_smol)
```

## Format

dataframe with 100 rows and 68 columns.
